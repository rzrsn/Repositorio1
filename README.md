# Repositorio1
# Sistema de Gestión de Estacionamiento

## Descripción

Este proyecto consiste en una simulación básica de un sistema de gestión de estacionamiento. Permite registrar la entrada y salida de vehículos, validar la disponibilidad de espacios y calcular tarifas en función del tiempo de estancia.

El objetivo es representar la lógica fundamental de operación de un estacionamiento automatizado.

## Funcionalidades

* Registro de entrada de vehículos
* Generación de boletos mediante placa
* Validación de espacios disponibles
* Cálculo de tarifas por tiempo
* Registro de salida de vehículos

## Arquitectura

El sistema sigue una estructura lógica simplificada:

* Entrada: simulada mediante funciones
* Lógica de negocio: validaciones, cálculos y control de estado
* Salida: mensajes en consola

## Tecnologías

* JavaScript (Node.js)
* Sin dependencias externas
